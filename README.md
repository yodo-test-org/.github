# .github

This repository contains organization-wide GitHub profile and community health files for yodo-test-org.

## Files

- profile/README.md: organization profile shown on the GitHub organization overview page
- SECURITY.md: default security policy for repositories without their own policy
- CONTRIBUTING.md: default contribution guidance
- CODE_OF_CONDUCT.md: default code of conduct
- SUPPORT.md: default support guidance
- PULL_REQUEST_TEMPLATE.md: default pull request template
- ISSUE_TEMPLATE/config.yml: default issue creation guidance

Repository-specific files take precedence over these defaults.
