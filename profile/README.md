# yodo-test-org

`yodo-test-org` は、GitHub Organization の機能、セキュリティ設定、共通運用ルールを検証するための Organization です。

## この Organization で扱うもの

- GitHub Organization 機能の検証
- `.github` リポジトリによる共通ファイル管理
- `SECURITY.md` の継承・上書き挙動の確認
- issue / pull request / contribution の運用ルール検証

## リポジトリ運用の基本方針

各リポジトリには、できるだけ次の情報を含めます。

- リポジトリの目的
- セットアップ方法
- 基本的な使い方
- メンテナンス方針
- セキュリティ報告方法

## Organization 共通ファイル

共通の security policy や contribution guide は、次のリポジトリで管理しています。

https://github.com/yodo-test-org/.github

## SECURITY.md の学習

`SECURITY.md` の Organization 共通設定とリポジトリ固有設定の違いは、次の学習用リポジトリで確認できます。

https://github.com/yodo-test-org/org-security-md-lab

## セキュリティ報告

セキュリティ上の問題は public issue では報告しないでください。報告手順は [SECURITY.md](https://github.com/yodo-test-org/.github/blob/main/SECURITY.md) を確認してください。
